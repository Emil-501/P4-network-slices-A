环境要求：

a.bmv2

## 注意：

**1：请将demo复制至目录`bmv2/target/`下运行！**

```
cp -r [demo_name] [your_dir]/bmv2/target
```

**2：在每一次退出mininet之后，请执行：**

```
sudo mn -c
```

**3：在编写commands.txt的过程中，请不要加入多余的空行，这会导致`DUPLICATE_ENTRY`错误。**
